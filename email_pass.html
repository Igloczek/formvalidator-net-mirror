Access Control For Your Web Site

Now you can easily secure access to areas of you web site
and require users to login with a user id and an optionally password.
Use of a password is optional.

Only a few lines of code in your html does it all.
Works in any web site, any where.
No server programming!
click here to learn how

To use The Form Validator access control feature on your site you must register and then access the management screen --> secure.formvalidator.net

The management system allows you to specify "allow-all" access or "deny-all" to your web form login.   In the "allow-all" scenario you manage a list of login ids that will be specifically blocked from submitting the form and gaining access.  Everyone else is allowed in.  In the "deny-all' scenario you manage a list of login ids that will specifically be granted access.   Everyone else is blocked out.  In both scenarios the utilization of additional password protection is optional.
